## Plant.id
식물의 이미지로 식물의 학명, 정확도 등을 판단할 수 있는 api

## 농사로
농촌 진흥청 농사로에서 실내 정원 식물의 정보를 제공. 이를 받아서 HellomyPlant 데이터베이스에 마이그레이션
